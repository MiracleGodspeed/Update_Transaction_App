# Update_Transaction_App


The application adopted a code first model entity approach.

Built with .NET Core 3.1..




USAGE

On successful run, the user is presented with short registration form, where he/she fills in details respectively.
The system generates a Wallet ID for the user which the user uses to access their account.


The link 'Manage transaction' presents the user with an input where he/she is expected to slot in their wallet addresses for transactions mangement
such as deposit, withrawal.
